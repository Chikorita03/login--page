# Login Page (HTML)

A simple login page built using semantic HTML.

## Features
- Username and password input fields
- HTML form with POST method
- Client-side validation using `required`
- Accessible labels using `for` and `id`

## Technologies Used
- HTML5

## Future Improvements
- Add CSS styling
